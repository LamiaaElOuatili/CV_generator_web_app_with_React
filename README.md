# CV_generator_web_app_with_React
This repository contains the source code and related files for a group project for a web application using JavaScript React to generate CVs.

# Usage
To run the application locally, follow these steps:

Clone the repository using the command git clone https://github.com/username/repository.git.

Install the necessary dependencies by running **npm install** in the root directory of the project.

Once the dependencies have been installed, start the development server by running **npm start**.

The application can now be accessed by navigating to http://localhost:3000 in a web browser.
